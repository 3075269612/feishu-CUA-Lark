# 飞书快捷键

## 全局快捷键

### 导航
- **Ctrl+K**：全局搜索（搜索消息、联系人、文档、日程）
- **Ctrl+1**：切换到消息模块
- **Ctrl+2**：切换到通讯录
- **Ctrl+3**：切换到日历
- **Ctrl+4**：切换到云文档
- **Ctrl+5**：切换到工作台
- **Ctrl+Tab**：切换到下一个会话
- **Ctrl+Shift+Tab**：切换到上一个会话

### 窗口操作
- **Ctrl+W**：关闭当前会话
- **Ctrl+Shift+W**：关闭所有会话
- **Ctrl+N**：新建会话/文档
- **Ctrl+,**：打开设置
- **Ctrl+Q**：退出飞书
- **F11**：全屏/退出全屏

### 消息操作
- **Ctrl+Enter**：发送消息（默认设置）
- **Enter**：换行（默认设置）
- **Ctrl+Shift+V**：粘贴为纯文本
- **Ctrl+F**：在当前会话中搜索
- **Ctrl+Shift+F**：全局搜索消息

## 消息编辑快捷键

### 文本格式
- **Ctrl+B**：加粗
- **Ctrl+I**：斜体
- **Ctrl+U**：下划线
- **Ctrl+Shift+X**：删除线
- **Ctrl+Shift+C**：代码块
- **Ctrl+Shift+K**：插入链接

### 消息操作
- **Ctrl+Z**：撤销
- **Ctrl+Y**：重做
- **Ctrl+A**：全选
- **Ctrl+C**：复制
- **Ctrl+V**：粘贴
- **Ctrl+X**：剪切

### 特殊输入
- **@**：提及用户
- **#**：提及话题
- **:**：插入表情（输入 `:smile` 等）
- **/**：快捷命令（部分场景）

## 云文档快捷键

### 文档编辑
- **Ctrl+B**：加粗
- **Ctrl+I**：斜体
- **Ctrl+U**：下划线
- **Ctrl+K**：插入链接
- **Ctrl+/**：显示快捷键列表
- **Ctrl+Shift+8**：插入无序列表
- **Ctrl+Shift+7**：插入有序列表

### 文档操作
- **Ctrl+S**：保存（云文档自动保存，此快捷键可能无效）
- **Ctrl+P**：打印
- **Ctrl+F**：查找
- **Ctrl+H**：替换

### 表格操作（电子表格）
- **Ctrl+C**：复制单元格
- **Ctrl+V**：粘贴单元格
- **Ctrl+X**：剪切单元格
- **Ctrl+Z**：撤销
- **Ctrl+Y**：重做
- **Tab**：移动到下一个单元格
- **Shift+Tab**：移动到上一个单元格
- **Enter**：移动到下方单元格

## 日历快捷键

### 视图切换
- **D**：切换到日视图
- **W**：切换到周视图
- **M**：切换到月视图
- **T**：回到今天

### 日程操作
- **C**：创建新日程
- **N**：创建新日程（同 C）
- **E**：编辑选中的日程
- **Delete**：删除选中的日程

### 导航
- **←**：前一天/周/月
- **→**：后一天/周/月
- **↑**：向上滚动
- **↓**：向下滚动

## 视频会议快捷键

### 会议控制
- **Ctrl+D**：静音/取消静音
- **Ctrl+E**：开启/关闭摄像头
- **Ctrl+Shift+S**：共享屏幕
- **Ctrl+Shift+R**：开始/停止录制
- **Ctrl+Shift+H**：举手/放下手

### 会议导航
- **Ctrl+Shift+P**：打开参会人列表
- **Ctrl+Shift+C**：打开聊天面板
- **Ctrl+Shift+M**：打开更多设置

## 使用建议

### 测试场景
1. **优先使用快捷键**：快捷键比鼠标点击更稳定、更快速
2. **组合使用**：例如 `Ctrl+K` 搜索 + `Enter` 打开第一个结果
3. **验证快捷键生效**：通过 OCR 或 VLM 验证快捷键执行结果

### 常见问题
- **快捷键冲突**：某些快捷键可能与系统或其他软件冲突
- **输入法影响**：中文输入法状态下某些快捷键可能无效，切换到英文输入法
- **焦点问题**：确保飞书窗口在前台且焦点正确

### 自动化建议
- **Ctrl+K 全局搜索**：最常用的快捷键，可以快速定位任何内容
- **Ctrl+1/2/3/4**：快速切换模块，比点击侧边栏更可靠
- **Ctrl+Enter**：发送消息，避免点击发送按钮
- **Escape**：关闭弹窗，通用的取消操作

## 快捷键优先级

在自动化测试中，建议按以下优先级使用快捷键：

1. **高优先级**（优先使用）：
   - Ctrl+K（全局搜索）
   - Ctrl+1/2/3/4（模块切换）
   - Ctrl+Enter（发送消息）
   - Escape（关闭弹窗）

2. **中优先级**（可选使用）：
   - Ctrl+F（搜索）
   - Ctrl+N（新建）
   - Ctrl+W（关闭）

3. **低优先级**（谨慎使用）：
   - 文本格式快捷键（可能受输入法影响）
   - 视图切换快捷键（可能不稳定）

## 验证要点

- 快捷键按下后，检查预期的 UI 变化
- 使用 OCR 验证文本变化（如加粗、斜体）
- 使用 VLM 验证界面切换（如模块切换）
- 记录快捷键执行时间和结果
