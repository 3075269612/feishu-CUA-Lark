# IM Skill

## Common Entrypoints

- The message module is usually in the left sidebar.
- Search can be used to find a test chat by name.

## Send Text Flow

1. Open messages.
2. Search for the allowed test chat.
3. Open the chat.
4. Focus the input box.
5. Paste the prepared text.
6. Press Enter or click send.
7. Verify the latest message bubble and absence of a failed-send marker.
