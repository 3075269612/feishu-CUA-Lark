# Docs 技能

## 常见入口
- 左侧栏"云文档"图标
- 顶部搜索框搜索文档
- 快捷键：Ctrl+4 快速切换到云文档

## 创建文档流程
1. 打开云文档模块
2. 点击"新建"按钮（通常在右上角）
3. 选择文档类型：
   - **文档**：富文本编辑器，类似 Word
   - **电子表格**：类似 Excel
   - **多维表格**：数据库视图
   - **思维笔记**：大纲笔记
4. 输入文档标题（**必须包含 `CUA-Lark` 等测试关键词**）
5. 编辑文档内容
6. 文档自动保存，无需手动保存
7. 验证文档出现在文档列表中

## 编辑文档流程
1. 在文档列表中找到目标文档
2. 单击文档打开编辑器
3. 编辑标题或正文内容
4. 使用工具栏格式化文本（加粗、斜体、标题等）
5. 插入图片、表格、链接等元素
6. 文档自动保存
7. 验证修改生效

## 分享文档流程
1. 打开目标文档
2. 点击右上角"分享"按钮
3. 选择分享方式：
   - **复制链接**：生成分享链接
   - **添加协作者**：邀请特定用户编辑
   - **设置权限**：可查看/可编辑/可评论
4. 确认分享设置
5. 验证分享链接可访问

## 删除文档流程
1. 在文档列表中找到目标文档
2. 右键点击文档，选择"删除"
3. 确认删除操作
4. 文档移动到回收站
5. （可选）在回收站中彻底删除

## 常见文档类型
- **文档**：适合写报告、方案、笔记
- **电子表格**：适合数据分析、表格整理
- **多维表格**：适合项目管理、数据库
- **思维笔记**：适合大纲、知识管理

## 常见异常
- 文档未保存：云文档通常自动保存，检查网络连接
- 权限不足：无法编辑他人文档，需要对方授予编辑权限
- 文档加载慢：大文档或网络慢时可能需要等待
- 协作冲突：多人同时编辑时可能出现冲突，刷新页面解决

## 验证要点
- 文档标题正确显示
- 文档内容完整保存
- 文档出现在正确的文件夹中
- 分享链接可访问（如果设置了分享）
- 协作者可以看到文档（如果添加了协作者）

## 与其他产品联动
- 从 IM 分享文档：在聊天中发送文档链接
- 从文档跳转到 IM：@提及用户时可以直接发起聊天
- 日历日程中可以附加文档链接

## 安全边界
- **测试文档必须包含 `CUA-Lark` 关键词**，避免污染真实文档库
- **测试文档必须创建在指定的测试文件夹中**
- 测试完成后应清理创建的文档
- 不要分享测试文档给真实用户
- 不要在真实项目文档中进行测试操作

## 常用快捷键
- Ctrl+B：加粗
- Ctrl+I：斜体
- Ctrl+U：下划线
- Ctrl+K：插入链接
- Ctrl+/：显示快捷键列表
